# 羊了个羊 - 堆叠式三消H5游戏

基于 React + Tailwind CSS 构建的类《羊了个羊》堆叠式三消游戏。

## 功能特性

- 🎮 **核心游戏机制**：模仿《羊了个羊》的堆叠式三消玩法
- 🧱 **多层堆叠系统**：瓦片以多层堆叠方式分布，上层瓦片遮挡下层瓦片
- 🎁 **盲盒系统**：左右两侧各有一个盲盒，内含堆叠瓦片，只有最前面瓦片可见
- 📥 **槽位管理**：底部7个槽位，用于存放选中的瓦片
- ✨ **消除逻辑**：槽位中出现3个相同瓦片时自动消除
- 🏆 **胜利/失败条件**：清除所有瓦片获胜，槽位填满且无法消除时失败
- 🎨 **现代化UI**：使用 Tailwind CSS 构建的响应式、可爱风格界面
- 🎭 **视觉反馈**：被遮挡瓦片变暗，激活瓦片高亮，平滑动画效果

## 技术栈

- **React 18**：函数组件和 Hooks
- **Vite**：构建工具和开发服务器
- **Tailwind CSS**：样式框架
- **原生 JavaScript**：游戏逻辑实现

## 游戏玩法

1. **目标**：清除场上所有瓦片（包括盲盒中的瓦片）
2. **操作**：
   - 点击主游戏区中未被遮挡的瓦片，将其移动到下方槽位
   - 点击左右两侧的盲盒，取走最前面的瓦片
3. **消除规则**：当槽位中出现3个相同的瓦片时，它们会自动消除
4. **失败条件**：当槽位被填满（达到7个）且无法触发消除时，游戏结束

## 核心算法

### 洗牌算法
- 确保生成的瓦片总数是3的倍数
- 使用 Fisher-Yates 洗牌算法随机排列瓦片
- 每种类型至少生成3个，确保游戏可解

### 碰撞检测算法
- 矩形碰撞检测判断瓦片重叠
- 按层级从高到低排序，计算遮挡关系
- 只有未被遮挡的瓦片处于激活状态（可点击）

### 消除判断算法
- 统计槽位中每种类型瓦片的数量
- 当某种类型数量达到3时触发消除
- 消除后重新计算游戏状态

## 实现细节

### 瓦片数据结构
```javascript
{
  id: string,          // 唯一标识
  type: string,        // 图案类型：'🐑', '🌽', '🥕', '🔔', '🍎', '🍌', '🍇', '🍒'
  layer: number,       // 层级，越大越在上层
  x: number,          // 水平位置（百分比）
  y: number,          // 垂直位置（百分比）
  width: number,      // 瓦片宽度
  height: number,     // 瓦片高度
  isBlocked: boolean, // 是否被遮挡
  isSelected: boolean, // 是否被选中
  isVisible: boolean   // 是否可见（用于盲盒）
}
```

### 状态管理
使用自定义 Hook `useGameState` 管理游戏状态，包括：
- 瓦片数组、盲盒数组、槽位数组
- 分数、步数、游戏状态
- 点击事件处理函数
- 消除检测和游戏状态检查

### 动画效果
- CSS过渡动画实现瓦片移动
- 关键帧动画实现消除效果
- 被遮挡瓦片使用 `filter: brightness(0.5)` 视觉区分
